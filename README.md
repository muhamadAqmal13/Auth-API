# Auth API

Auth API adalah sebuah project pembelajaran saya, dimana Auth API ini digunakan
untuk mengauthentikasi sistem registration, login dan logout

Dibuat dengan:

-   MongoDB
-   ExpressJS
-   JWT
-   NodeJS
